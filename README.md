# Amazon EventBridge Scheduler (amazon-eventbridge-scheduler)
Amazon EventBridge Scheduler is a serverless scheduler that allows you to create, run, and manage tasks from one central, managed service. It automatically delivers tasks to over 270 AWS services and over 6,000 API operations, removing the need to manage separate scheduling infrastructure.

### Amazon EventBridge Scheduler API
The Amazon EventBridge Scheduler API provides programmatic access to create and manage schedules, schedule groups, and targets for automated task execution across AWS services.
